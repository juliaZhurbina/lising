# Telegram Bot с GigaChat API для управления заявками

Бот для автоматического изменения статусов заявок на основе естественного языка с использованием GigaChat API.

## Возможности

### Основные функции

1. **Изменение статуса заявки**
   - Изменение статуса по запросу на естественном языке
   - Интерактивное подтверждение изменений
   - Пример: "измени статус заявки Z12345 на 15 ноября"

2. **Поиск информации по заявке**
   - Поиск заявок по ID, клиенту, автомобилю
   - Получение полной информации о заявке
   - История изменений и статусов
   - Примеры: "найди заявку Z25-1869607", "покажи информацию о заявке Z12345"

3. **Генерация контекста для звонков и встреч**
   - Автоматическая подготовка структурированного контекста
   - Анализ истории заявки через GigaChat
   - Рекомендации для эффективной коммуникации
   - Примеры: "подготовь контекст для звонка по заявке Z12345", "что нужно знать перед встречей"

4. **Управление задачами**
   - Создание задач по заявкам
   - Напоминания о важных датах и событиях
   - Отслеживание выполнения задач
   - Примеры: "создай задачу позвонить клиенту заявки Z12345 завтра", "напомни о встрече"

### Технические особенности

- Интеллектуальное понимание естественного языка через GigaChat API
- Работа с базой данных заявок в Excel (2180+ записей)
- Интерактивный интерфейс с кнопками и командами
- Структурированные ответы и форматирование данных

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скопируйте `.env.example` в `.env` и заполните необходимые данные:
```bash
copy .env.example .env
```

3. Получите токен Telegram бота у [@BotFather](https://t.me/BotFather)

4. Получите учетные данные GigaChat API на [GigaChat](https://developers.sber.ru/gigachat)

5. Запустите бота:
```bash
python main.py
```

## Использование

1. Найдите бота в Telegram и отправьте `/start`
2. Используйте команды или естественный язык для работы с заявками:

### Команды бота:
- `/start` - приветствие и инструкция
- `/help` - справка по использованию
- `/tasks` - список ваших активных задач
- `/search <запрос>` - быстрый поиск заявок

### Примеры запросов:

**Изменение статуса:**
- "измени статус заявки Z12345 на 15 ноября"
- "обнови дату заявки Z25-1869607 на 20.11"
- "поменяй статус Z12345 на 15 ноября"

**Поиск информации:**
- "найди заявку Z25-1869607"
- "покажи информацию о заявке Z12345"
- "какой статус у заявки Z25-1869607?"

**Генерация контекста:**
- "подготовь контекст для звонка по заявке Z12345"
- "что нужно знать перед встречей с клиентом заявки Z25-1869607?"
- "сгенерируй краткую справку по заявке Z12345"

**Работа с задачами:**
- "создай задачу позвонить клиенту заявки Z12345 завтра"
- "напомни мне о встрече по заявке Z25-1869607"
- "какие задачи у меня по заявке Z12345?"

3. Подтверждайте действия через интерактивные кнопки
